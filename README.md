# Wireframe Renderer

Wireframe Renderer is a lightweight 3D wireframe rendering tool written in C. Utilizing MiniLibX and OpenGL, it can load heightmaps, perform transformations, and render them as wireframes in real-time.

## Features

- Real-time rotation, panning, and zooming of 3D models
- Orthographic and perspective projection modes
- Lightweight and low-level using MiniLibX only for windowing and input

## Requirements

- macOS (potentially Linux if compiled correctly)
- Xcode (optional)
- Xcode Command Line Tools installed
- MiniLibX
- Frameworks (refer to the Makefile)

## Getting Started

### Clone the repository

### Build and Run (Xcode)

Open the project in Xcode

### Build and Run (Command Line)

Run make in the sourcecode root
